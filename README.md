# pkge
A package manager for proton-ge written in bash. <br>
It can list, remove and install the latest and greatest proton-ge versions fresh from GlouriousEggrolls github-repo.

## dependencies
- bash
- coreutils
- curl
- git
- jq
- tar
- wget
